# 实时天气查询与出行穿搭建议

> 支持实时温度、湿度、风力、空气质量查询，未来 7 天天气预报，智能穿衣建议及极端天气预警

## 前置依赖

```bash
pip install pandas
```

## 核心能力

### 能力1：询问所在城市

用 `web_search` 搜索实时天气数据，用 `web_fetch` 抓取气象网站详细预报（温度/湿度/降水/紫外线）。

### 能力2：联网查询实时天气（web_search，温度/湿度/风力/空气质量）

用 `web_search` 搜索实时天气数据，用 `web_fetch` 抓取气象网站详细预报（温度/湿度/降水/紫外线）。

### 能力3：提供未来

用 `web_search` 搜索实时天气数据，用 `web_fetch` 抓取气象网站详细预报（温度/湿度/降水/紫外线）。

### 能力4：根据天气给出穿衣建议

用 `web_search` 搜索实时天气数据，用 `web_fetch` 抓取气象网站详细预报（温度/湿度/降水/紫外线）。

### 能力5：遇到极端天气发出预警提示

用 `web_search` 搜索实时天气数据，用 `web_fetch` 抓取气象网站详细预报（温度/湿度/降水/紫外线）。

## 使用流程

### 步骤 1：收集用户需求

向用户确认以下信息（如果未主动提供）：
- 你在哪个城市？
- 查询哪天的天气？（今天/明天/本周）
- 是否有特殊活动（约会/面试/户外运动）？
- 衣橱偏好（简约/商务/运动/时尚）

### 步骤 2：检索外部信息

执行以下搜索获取真实数据：

```
web_search("[城市] 天气预报 今天")
web_search("[城市] weather forecast")
```

确保获取到以下资源：
- 天气数据链接
- 穿搭图片参考

### 步骤 3：运行脚本处理数据

```bash
python3 scripts/weather_outfit_advisor_tool.py run \
  --input "用户提供的输入" \
  --output "/path/to/output_file"
```

读取脚本输出的结果，确认数据处理成功。

### 步骤 4：生成最终产出

基于脚本输出和搜索到的资源，用 `write_to_file` 生成以下文件：

- **天气+穿搭建议文档**

输出格式要求：天气数据 + 穿搭建议 + 预警提示

### 步骤 5：汇总交付

向用户展示：
1. 生成的文件路径和内容摘要
2. 搜集到的资源链接列表
3. 关键发现和建议

## 注意事项

- 所有数据必须来自 `web_search` / `web_fetch` 的真实搜索结果，**严禁编造数据**
- 数据缺失时标注"数据不可用"而非猜测
- 报告必须保存为文件（`write_to_file`），不能只在对话中输出
- 建议结合人工判断使用，AI 分析仅供参考
