## 🚗 Day 1 - 周五 2026-03-20 深圳出发

- 从深圳出发
- **出发前加满油，检查车辆**

### 晚上 7:00
- 在**阳江服务区**或**茂名服务区**休息，吃晚餐

## 🏖️ Day 2 - 周六 2026-03-21 老街 + 贵州路市场 + 银滩 + 侨港夜市

### 上午

| 时间 | 活动 |
|------|------|
| 9:30 | 自然醒，在酒店吃早餐 |
| 10:30 | 前往**北海老街**，逛骑楼建筑，拍照 |
| 11:00 | 老街对面海滩散步看海 |
| 11:30 | 在老街吃**蟹仔粉**（15-25 元/碗） |

### 🍜 老街备选美食
- **李姨虾饼**（教堂对面）- 备选
- **椰椰大吉**（待确认是否有店）- 泰汁椰奶冻、芒果糯米饭
- **北海田螺店** - 三牛粉、叉烧螺蛳粉（备选）

### 中午

| 时间 | 活动 |
|------|------|
| 12:00 | 离开老街，前往**贵州路市场** |
| 12:30 | 到达贵州路市场，买海鲜 |
| 13:30 | 找附近饭店加工海鲜，吃午饭 |

### 🦐 海鲜采购清单
- 螃蟹 2-3 只
- 蛏子 1-1.5 斤
- 虾 1-1.5 斤
- 生蚝 6-10 个
- 鲍鱼 6-8 个
- 海胆 2-4 个
- 生菜 1 份

### 🍽️ 午餐 7 道菜
香辣蟹、炒生菜、葱姜炒蛏子、油焖虾、生蚝、蒸鲍鱼、海胆

### 下午

| 时间 | 活动 |
|------|------|
| 14:00 | 午饭结束 |
| 15:00 | 返回酒店休息 或 自由活动 |
| 17:00 | 前往**银滩** |
| 17:30 | 银滩游玩、游泳、拍照、玩沙 |
| 19:00 | 银滩看日落 |
| 19:30 | 前往**侨港风情街**（打车或开车约 15 分钟） |

### ⚠️ 银滩坐船提醒
- 可以坐船出海，**记得砍价！对半砍**
- 可以砍到**100 块两个人**坐一次

### 晚上 - 侨港风情街夜市

| 时间 | 活动 |
|------|------|
| 20:00 | 逛侨港风情街夜市，也可以去**侨港籺仔街** |
| 20:30 | **信记虾饼** |
| 21:00 | **榴莲芝士饼**（现烤的很香） |
| 21:30 | **墨鱼丸**（侨港特色小吃，记得吃） |
| 22:00 | **虾仁饼**（侨港特色，推荐尝试） |

## ⚠️ 重要提醒

1. 4 天 3 夜时间充裕，可以灵活调整，有些地方不去也没关系
2. **周六早上吃蟹仔粉，周日早上吃海鲜粉**
3. **贵州路市场中午去**，海鲜新鲜，多比价可以砍价
4. **加工费先问清楚**，大约 15-30 元一个菜
5. 下午 5 点到银滩，时间充裕，可以游泳玩沙
6. **银滩坐船可以砍价，对半砍，100 块两个人**
7. 银滩日落 3 月大约 6 点半到 7 点，**提前查好当天时间**
8. 侨港风情街晚上逛，夜市小吃多
9. **侨港夜市记得吃墨鱼丸、虾仁饼**
10. **糖水推荐试试甘蔗雷公根**，北海本地特色，比较甜
11. **周日中午糟粕醋火锅，包浆豆腐必点**
12. **桂小厨柠檬鸭必点**
13. 可以去新力广场、万达广场逛街
14. 备选美食很多，看时间和兴趣决定吃不吃
15. 越乡小厨可以点香草虾、捞汁小海鲜、海鲜砂锅粥
16. 杯杯乐可以吃牛肉番茄炒粉、鱼腐
17. 侨港籺仔街可以试试越南滴漏冰摇咖啡、越南卷粉
18. **友口福海鲜餐厅推荐**：沙虫刺身、梅子蒸魔鬼鱼、鱼肝、九层塔炒海豆芽、榄钱、炒鱿鱼、粉丝生蚝、沙蟹汁炒豆角
19. **返程下午 4 点出发，预计晚上 11 点到深圳，注意夜间驾驶安全**
20. **住宿提前预订**，周末可能紧张
21. **出发前查好北海天气预报**
22. **吃海鲜备好肠胃药**
